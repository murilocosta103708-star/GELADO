#  Sobre o Projeto

O **Gelado Intelligence** é um projeto de Business Intelligence voltado para empresas de distribuição de gelo, focado em:

- Controle operacional
- Eficiência logística
- Gestão financeira
- Inteligência industrial
- Monitoramento de perdas
- Análise de rentabilidade

O sistema transforma dados operacionais em indicadores estratégicos para auxiliar na tomada de decisões.

---

#  Objetivos Estratégicos

## ⚡ Otimização Energética
Monitorar consumo energético e eficiência produtiva.

##  Eficiência Logística
Melhorar ocupação da frota e reduzir custos operacionais.

##  Rentabilidade Real
Identificar margem de lucro por cliente, região e canal.

##  Controle de Perdas
Reduzir desperdícios e derretimento durante transporte e armazenagem.

#  Perguntas Respondidas

O sistema responde questões importantes como:

- Quanto foi faturado hoje?
- Qual produto mais vendido?
- Qual cliente gera maior lucro?
- Quanto gelo foi perdido?
- Qual região possui maior demanda?
- Qual rota é mais cara?
- O caminhão saiu com capacidade máxima?
